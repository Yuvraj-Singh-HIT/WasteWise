# **App Name**: WasteWise

## Core Features:

- Image Upload: Allow users to upload images of waste items.
- Waste Identification: Utilize computer vision to identify the type of waste in the image (e.g., plastic, paper, glass, organic).
- Segregation Guide: Provide users with instructions on how to properly segregate the identified waste item.
- Educational Content: Include informative content about the importance of waste segregation and recycling. Display this information through the app's UI.
- AI Confidence Scores: Communicate confidence scores alongside waste suggestions using computer vision AI.

## Style Guidelines:

- Primary color: Forest green (#388E3C) to symbolize environmental friendliness and cleanliness.
- Background color: Light beige (#F5F5DC) to provide a clean and neutral backdrop.
- Accent color: Sky blue (#3F51B5) to draw attention to important information.
- Body and headline font: 'PT Sans', a humanist sans-serif, provides a modern and approachable look, making the information easily accessible and encouraging engagement.
- Use clear and recognizable icons to represent different types of waste (e.g., plastic bottle, paper, glass).
- Simple, intuitive layout with a clear hierarchy of information. The image upload area should be prominent.
- Use subtle animations, such as smooth transitions, to provide feedback to the user.
